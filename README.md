# 日本史 一問一答チャレンジ

中学3年生・高校入試対策のための、日本史4択クイズアプリです。

## 特長

- **時代を選んで出題**：鎌倉・室町・江戸など、時代別に選べます（複数選択可）。
- **問題数を選択**：5〜50問から選べます。
- **まちがえたら正解するまで進めない**：正しい答えを選ぶまで次に進めません。
- **解説つき**：解答後に、その問題のポイントを解説します。
- **正答率と復習**：最後に正答率を表示し、まちがえた問題をふりかえれます。
- **スマホ対応**：スマートフォンでも快適に使えます。

## 使い方（ローカルで試す場合）

`fetch` で問題データを読み込むため、ファイルを直接開くのではなく簡易サーバーが必要です。

```bash
cd 勉強用アプリ作成
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## ファイル構成

```
index.html            アプリ本体
css/style.css         デザイン
js/app.js             プログラム
data/manifest.json    時代の一覧
data/<時代>.json       各時代の問題データ
```

## 問題データの形式（追加・編集用）

```json
{
  "era": "鎌倉時代",
  "questions": [
    {
      "id": "kamakura-001",
      "question": "問題文",
      "options": ["正しい答え", "選択肢2", "選択肢3", "選択肢4"],
      "answer": 0,
      "explanation": "解説文",
      "image": "（任意）画像URL",
      "imageCaption": "（任意）画像の説明"
    }
  ]
}
```

- `answer` は `options` の中で正解が何番目か（0から数える）。
- 選択肢の表示順はアプリ側で自動的にシャッフルされます。
- 新しい時代を追加したら `data/manifest.json` にも追記してください。
